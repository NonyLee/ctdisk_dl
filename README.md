# ctdisk_dl
城通网盘下载,多线程下载

下载文件用 -f\r\n
下载目录用 -d
文件分块 -s
同时多个文件 -c

optional arguments:
  -h, --help            show this help message and exit
  -d DIR, --dir DIR     download a directory
  -f FILE, --file FILE  download a file
  -s SPLIT, --split SPLIT
                        split a files to parts
  -c DL_CNT, --dl_cnt DL_CNT
                        download files concorrent
